## Описание REST api интерфейса
- **Метод**: POST
- **URL**: /api/auth/register
- **Входные параметры**:
  | Наименование | Тип | Обязательность | Ограничения |
  | :--- |:--- | :--- | :--- |
  | First Name | string | Да | Не указаны |
  | Last Name | string | Да | Не указаны |
  | UserName | string | Да | Уникальное |
  | Password | string | Да | пароль должен содержать как минимум один не буквенно-числовой символ, одну цифру ('0' - '9'), одну заглавную букву ('A' - 'Z'), одну строчную букву ('a' - 'z'), один специальный символ, а так же состоять из восьми и более символов |
  | reCaptchaToken | string | Да | Токен, полученный от виджета reCAPTCHA |

 - **Выходные данные при успешном ответе (код ответа: 201 Created)**:
```
{
  "id": 1,
  "firstName": "Ivan",
  "lastName": "Ivanov",
  "userName": "ivan"
}
```
- **Выходные данные при некорректном пароле (Код ошибки: 400 Bad Request):**
```
{
  "error": "Passwords must have at least one non alphanumeric character,
            one digit ('0'-'9), one uppercase ('A'-'Z'), one lowercase ('a'-'z'),
            one special character and Password must be eight characters or longer."
}
```
- **Выходные данные при существующем значении поля UserName (Код ошибки: 409 Conflict):**
```
{
  "error": "User exists!"
}
```
- **Выходные данные при непройденной reCaptcha (Код ошибки: 400 Bad Request):**
```
{
  "error": "Please verify reCaptcha to register!"
}
```
- **Пример запроса: **
```
POST /api/auth/register HTTP/1.1
Host: demoqa.com
Content-Type: application/json

{
  "firstName": "Ivan",
  "lastName": "Ivanov",
  "userName": "ivan",
  "password": "Str0ng!Pass",
  "recaptchaToken": "03AGdBq24..."
}
```
- **Пример ответа успешного сценария:**
```
{
  "id": 1,
  "firstName": "Ivan",
  "lastName": "Ivanov",
  "userName": "ivan"
}
```
- **Пример ответа сценария с ошибкой:**
```
{
  "error": "Validation failed",
  "details": [
    "Please verify reCaptcha to register!"
  ]
}
```
